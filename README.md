## 🌸Functionalities — Gamified Focus Flow Timer

### 🎯 Core Timer Features

1. **Pomodoro Study Timer**

   * Custom focus duration
   * Countdown display
   * Start / Pause / Reset controls

2. **Break Timer**

   * Custom break duration
   * Automatic switch after session completion
   * Separate focus & break modes

3. **Session Tracking**

   * Counts completed focus sessions
   * Displays current session number
   * Helps maintain study streak awareness

4. **Real-Time Clock**

   * Shows current system time continuously

---

## ✅ Task Management System

5. **Add Tasks**

   * User can create unlimited tasks
   * Each task can include custom reward minutes

6. **Mark Task as Completed**

   * Completed tasks show ✔ indicator
   * Completion automatically adds reward minutes to Break Bank

7. **Delete Tasks**

   * Remove selected tasks from list

8. **Task Reward System**

   * Example:

     * “Finish DSA sheet” → +10 break minutes
     * “Complete assignment” → +15 break minutes

---

## 🌸 Break Bank System

9. **Break Bank Storage**

   * Stores unused break minutes
   * Reward minutes from tasks also accumulate here

10. **Skip Break Logic**

* After a focus session ends:

  * App asks:

    > “Do you want to take a break?”
* User gets ~5 seconds to decide
* If skipped:

  * break duration is added to Break Bank
  * next focus session starts automatically

11. **Redeem Break Bank**

* User can later use stored break minutes
* Encourages productivity without losing rewards

---

## 🎮 Gamification Features

12. **Focus / Break Mode Indicator**

13. **Progress Bar**

* Live visual countdown progress

14. **Achievement Feeling**

* Session completion popups
